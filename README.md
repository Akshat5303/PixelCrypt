**Hide Secrets in Plain Sight.**

PixelCrypt is a web-based tool designed to perform **Image Steganography**. It allows users to securely hide secret text messages inside the pixels of an image. The visual difference is undetectable to the human eye, but the data can be retrieved using the decryption feature.
